### 🚨 CRITICAL FIX: EQ Line Z-Order (NOW VISIBLE)

**Root Cause:** The EQ (Equilibrium) line was drawn INSIDE the same code block as the P&D zone boxes. In Pine Script, filled boxes painted over the line because both were created in the same rendering pass. The EQ line was effectively invisible on the chart.

**Solution Applied:**
1. **Separated EQ line into its own dedicated block** — moved AFTER all `box.new()` calls (P&D zones AND S&D zones)
2. **Delete-and-redraw pattern** — `line.delete(pdEqLine)` → `line.new()` on every bar ensures the line is ALWAYS the last visual element rendered
3. **Width increased:** 4 → **5** (maximum visibility)
4. **Color:** `color.white` (pure white, 0% transparency)
5. **Style:** `line.style_solid`, `extend.right`
6. **EQ Label:** Solid black background (`color.new(color.black, 0)` — 0% transparency), `size.normal`, `label.style_label_left`

**Before:** EQ line invisible (hidden behind zone fills)
**After:** EQ line is the TOPMOST visual element on the chart
