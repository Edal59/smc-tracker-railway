Quick Setup Guide
  1. Copy the Webhook URL above
  2. In TradingView: Create Alert → Webhook URL → Paste
  3. Copy the Alert Message Template into the alert body
  4. Make sure your API key is in the alert body JSON
  5. Click Dashboard to view your performance!
