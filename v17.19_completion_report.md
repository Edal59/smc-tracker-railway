## Visual Change Summary

### FIX 1: P&D Equilibrium Line (CRITICAL)

| Property | v17.18 | v17.19 |
|----------|--------|--------|
| Label tracking | `label.new()` (untracked, accumulates) | `var label pdEqLabel` + `label.delete()` |
| Label text | `"EQ"` | `"  EQ  "` (padded for visibility) |
| Label background | `color.new(color.black, 0)` (opaque) | `color.new(color.black, 20)` (slight transparency) |
| Line | width=4, white, solid, extend.right | Unchanged |

### FIX 2: HUD High-Contrast Background

| Property | v17.18 | v17.19 |
|----------|--------|--------|
| Background | `color.new(color.black, 70)` (30% opacity) | `color.new(color.black, 40)` (60% opacity) |
| Frame border | None (frame_width=0) | `color.new(color.white, 80)`, frame_width=1 |

### FIX 3: HUD Float Mode

| Property | Details |
|----------|---------|
| New input | `hudModeInput` ‚Äî "Fixed" (default) / "Hover" |
| Fixed mode | Always 60% opacity |
| Hover inactive | 15% opacity (transp 85) |
| Hover active | 60% opacity ‚Äî triggers on `barstate.islast` or recent signal (<5 min) |

### FIX 4: HUD Size Optimization

| Property | v17.18 | v17.19 |
|----------|--------|--------|
| Content text size | `size.normal` | `size.small` (~15% smaller) |
| Header text size | `size.normal` | `size.normal` (unchanged) |
| Label names | Full names | Shortened (Align, KZ, POI, Confl, AMD, Trade, Entry, Stage, Alert, Signals) |

### FIX 5: Text Priority Hierarchy

| Role | Color | Usage |
|------|-------|-------|
| Header labels | `color.white` | Row labels (Bias, Align, KZ, etc.) |
| Active values | `color.lime` | Aligned state, positive signals |
| Neutral | `color.new(color.gray, 30)` | Inactive/static values |
| Warning | `color.orange` | Caution states |
| Invalidated | `color.red` | Conflicts, errors |
